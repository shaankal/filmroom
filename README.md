## What Is It

Film Room is an iOS app that trains football IQ through AI-generated X's & O's scenarios. Users read a formation, identify the coverage, and make the correct football decision — just like a real player or coach would in a film session.

Every scenario is generated by Claude, rendered as a live diagram, and followed by a teaching explanation. The app adapts to each user's knowledge level over time.

---

## Core Loop

```
Select position & difficulty
        ↓
Claude generates a play scenario (formation, coverage, down & distance)
        ↓
App renders X's & O's diagram
        ↓
Claude generates question + 4 answer choices
        ↓
User picks an answer
        ↓
Explanation revealed — teaches the underlying concept regardless of right/wrong
        ↓
Concept tagged → feeds adaptive learning system
```

## Difficulty Tiers

| Tier | Target User | Diagram | Question | Hints |
|---|---|---|---|---|
| 🟢 Rookie | Casual fan | Full labels + coverage named | Plain English, full context | Always shown |
| 🔵 Starter | Fantasy player / avid fan | Labels shown, coverage hidden | Terminology used, no coverage named | Hidden |
| 🟠 Veteran | Played the game | Simplified labels, no formation name | Concise, coach-room language | Hidden |
| 🟣 Coach | Coaches / analysts | No labels, raw diagram only | Minimal — "Your read?" | Hidden |

Difficulty is earned through performance — users start at their calibrated tier and progress by hitting accuracy thresholds. Coach difficulty is never self-selectable.

---

## Question Categories

Each position has distinct question types. Examples:

- **QB** — Where to throw, Coverage ID, Hot route / audible, Progression read, What happens if wrong
- **WR** — Release decision, Route conversion, Leverage & positioning
- **RB** — Gap read, Cutback lane, Blitz pickup
- **CB** — Press vs bail, Leverage technique, Route recognition
- **LB** — Gap assignment, Pass drop zone, Blitz timing
- **OC** — Play call, Personnel grouping, Motion & alignment
- **DC** — Coverage call, Blitz package, Personnel adjustment

Users can also **request custom scenarios** by concept — e.g. "give me a choice route scenario" or "practice Cover 3 seam reads" — and Claude generates it at their current difficulty.

---

## Adaptive Learning System

- Every scenario is tagged with a **concept tag** (e.g. `cover_3_seam`, `a_gap_integrity`)
- Tags with below 60% accuracy are surfaced as **weak areas**
- **Drill mode** generates 5 focused scenarios around a single weak concept
- **Difficulty auto-suggest** nudges users up or down based on recent accuracy trends
- Explanation panel teaches the concept after every answer — learning happens whether the user is right or wrong

## Onboarding & Calibration

1. **Account creation** — Sign in with Apple / Google / email. No guest mode.
2. **Position select** — Choose primary position
3. **Background questionnaire** — Football background, how they engage with the game, main goal
4. **Calibration** — 5 questions that scale in difficulty (basic → expert). Questions get progressively harder so guessing doesn't carry users to higher tiers.
5. **Placement result** — App explains starting difficulty and why
6. **Tutorial play** — One hardcoded, hand-crafted scenario per position. Not AI generated — guarantees a great first impression.
7. **Land on Train Hub** — First AI-generated session ready to go
